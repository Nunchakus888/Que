## 模仿Vue实现的简单MVVM框架
非生产力工具。

### 已实现：
![预览](http://static.wittsay.cc/que-test1.gif)

 - 双向绑定
 - 模板字符串解析
 - 原生事件绑定
 - 事件解绑
 - 模板字符串携带表达式
 - 生命周期
